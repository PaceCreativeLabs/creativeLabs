Tools

Some tools you will be using




  • GitHubVersion control

    GitHub is the equivalent of Google docs, but for code. And consequently you have to know how to use the version control system GIT. Putting together these two tools allow teams to work remoteley.

  • Text EditorsSublime / Notepad++

    Many of us use SublimeText for all of its features. If you prefer a notepad feel when programming then we at the very least suggest you use Notepad++ for its ability to format various programming languages.

  • TerminalYour preference

    Since GIT is meant to be used in the command prompt you will need to know how to navigate your way through your computer in a terminal whether that's on Windows or OSX.
